# Data Acquisition - PCB Shock Test Damage
Run instructions
1. Set the individual "SetBK" VI's to their frequency. Open the DataAcquisition.vi block diagram. In the case frame containing the various SetBK VI's, double click each VI and set the paramaters to their specified frequency, as well as setting the primary and secondary paramaters to Z and theta, respectively.
2.  In the front panel of the DataAcquisition.vi file, set the file path where the output files should be placed. Include a .lvm file name for the dataset to be printed to, ie (C:\Users\Person\Documents\Data\data.lvm).
3. in the front panel of the DataAcquisition.vi file, set the number of drops the program will be recording.
